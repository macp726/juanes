📐 Álgebra

¿Qué es factorizar?

Factorizar es descomponer una expresión algebraica en un producto de factores más simples. Es como deshacer una multiplicación.

Ejemplo numérico: 6 = 2 × 3  → factorizamos el número 6 en sus factores primos.

En álgebra hacemos lo mismo pero con expresiones que contienen variables y exponentes.

Diferencia de Cuadrados

Fórmula fundamental

a² − b² = (a + b)(a − b)

¿Por qué funciona? Verificamos multiplicando el lado derecho:

1
Aplicamos FOIL: (a+b)(a−b) = a·a − a·b + b·a − b·b
2
Simplificamos: = a² − ab + ab − b²
3
Los términos medios se cancelan: = a² − b² ✓
Ejercicio 1 – Selección múltiple

Una expresión algebraica a² − b² corresponde a una diferencia de cuadrados. Se factoriza como:

  • A (a + b)²
  • B (a + b)(a − b)
  • C (a − b)²
  • D a(a − b)

Justifica tu respuesta en el espacio siguiente:

💡 Justificación de la respuesta B:
Elegí B porque (a+b)(a−b) es exactamente la fórmula de la diferencia de cuadrados.
Verificación multiplicando FOIL:
(a+b)(a−b) = a² − ab + ab − b² = a² − b² ✓
Los términos medios (+ab y −ab) se cancelan entre sí, dejando solo la diferencia de cuadrados.
Ejercicio 2 – Factorizar

Factoriza las siguientes expresiones usando diferencia de cuadrados:

  • a) x² − 36
  • b) 49a² − 64b²
  • c) 25m² − 4n²
💡 Soluciones:
a) x² − 36: √x²=x, √36=6 → (x + 6)(x − 6)
b) 49a² − 64b²: √49=7, √64=8 → (7a + 8b)(7a − 8b)
c) 25m² − 4n²: √25=5, √4=2 → (5m + 2n)(5m − 2n)
Recuerda verificar: multiplica los factores y debes recuperar la expresión original.

Ejemplos resueltos paso a paso

Ejemplo 1: x² − 36
1
Identificar: a² = x² → a = x  |  b² = 36 → b = 6
2
Verificar raíces exactas: ✓ √x² = x, √36 = 6
3
Aplicar fórmula:
x² − 36 = (x + 6)(x − 6)

Verificación: (x+6)(x−6) = x²−6x+6x−36 = x²−36 ✓

Ejemplo 2: 49a² − 64b²
1
a² = 49a² → a = 7a  |  b² = 64b² → b = 8b
2
Raíces: √49 = 7 ✓  √64 = 8 ✓
49a² − 64b² = (7a + 8b)(7a − 8b)
Ejemplo 3: 25m² − 4n²
1
a = 5m  |  b = 2n
25m² − 4n² = (5m + 2n)(5m − 2n)

Casos que NO son diferencia de cuadrados

  • x² + 36  →  NO es suma, no diferencia
  • x² − 7  →  NO 7 no tiene raíz cuadrada exacta
  • x³ − 9  →  NO x³ no tiene raíz cuadrada exacta
Ejercicio 3 – Modelo de áreas

La región amarilla de la izquierda se reorganiza para formar la región amarilla de la derecha. Escribe la igualdad de áreas que representa:

a² − b² = (a + b)(a − b)
Región amarilla = a² − b²
⟹
Rectángulo = (a+b)(a−b)

Geométricamente: un cuadrado grande (a×a) al que le quitamos un cuadrado pequeño (b×b). Esa región puede reorganizarse en un rectángulo de dimensiones (a+b) × (a−b). Por eso la igualdad es geométricamente cierta.

Ejercicio 4 – Representación gráfica

Representa gráficamente la siguiente factorización:

9x² − 4 = (3x + 2)(3x − 2)

Dibuja o describe el modelo de áreas que muestra esta igualdad. Identifica a = 3x y b = 2.

💡 Modelo de áreas para 9x² − 4:
• Identificar: a = 3x (porque √9x² = 3x), b = 2 (porque √4 = 2)
• Cuadrado grande: lados de 3x, área = 9x²
• Cuadrado pequeño: lados de 2, área = 4 (se quita de una esquina)
• La región restante (9x²−4) se puede reorganizar en un rectángulo de (3x+2) × (3x−2)
• Verificación: (3x+2)(3x−2) = 9x² − 6x + 6x − 4 = 9x² − 4 ✓
Trinomio Cuadrado Perfecto (TCP)

Definición y fórmulas

Un Trinomio Cuadrado Perfecto tiene la forma a² ± 2ab + b² y es el resultado de elevar al cuadrado un binomio.

(a + b)² = a² + 2ab + b²
(a − b)² = a² − 2ab + b²

Condiciones para identificarlo:

  • El primer término debe tener raíz cuadrada exacta → a²
  • El tercer término debe tener raíz cuadrada exacta → b²
  • El término del medio debe ser el doble del producto de esas raíces → 2ab

Si las 3 condiciones se cumplen → ✓ Es TCP

Pasos para identificar un TCP

1
¿El primer término tiene raíz cuadrada exacta? → identifica a
2
¿El tercer término tiene raíz cuadrada exacta? → identifica b
3
¿El término del medio es igual a 2·a·b?

Si las 3 respuestas son SÍ → es TCP y puedes factorizar como (a ± b)².

Ejemplos resueltos

Ejemplo 1: x² + 10x + 25
1
√x² = x ✓ → a = x
2
√25 = 5 ✓ → b = 5
3
¿2·x·5 = 10x? ✓

→ Es TCP (suma) ✓

x² + 10x + 25 = (x + 5)²
Ejemplo 2: 4x² + 8x + 15
1
√4x² = 2x ✓ → a = 2x
2
√15 = 3.87... ✗ NO tiene raíz exacta

→ ✗ NO es TCP (el tercer término falla)

Ejercicio – Identificar TCP

Indica si cada expresión es un Trinomio Cuadrado Perfecto y justifica usando los 3 pasos:

  • a) x² + 10x + 25
  • b) 4x² + 8x + 15
💡 Solución:
a) x² + 10x + 25 → SÍ es TCP:
Paso 1: √x² = x ✓ → a = x
Paso 2: √25 = 5 ✓ → b = 5
Paso 3: 2·x·5 = 10x ✓ (coincide con el término medio)
Factorización: (x + 5)²

b) 4x² + 8x + 15 → NO es TCP:
Paso 1: √4x² = 2x ✓ → a = 2x
Paso 2: √15 ≈ 3.87 ✗ NO tiene raíz cuadrada exacta
→ Falla la condición 2, por lo tanto NO es TCP.
Ejercicio – Factorizar TCP

De la lista anterior, factoriza los que correspondan a trinomios cuadrados perfectos. Muestra el procedimiento completo:

💡 Solo el inciso a) es TCP:
x² + 10x + 25 = (x + 5)²
Verificación: (x+5)² = x² + 2(x)(5) + 5² = x² + 10x + 25 ✓

4x² + 8x + 15 NO es TCP porque √15 no es exacto. No se puede factorizar como binomio al cuadrado.
